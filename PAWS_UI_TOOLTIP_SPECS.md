# Техническое задание для разработки Paws-UI (Tooltip System)

**Контекст:**
Библиотека компонентов используется как в основном приложении (Electron), так и внутри изолированных iframe'ов плагинов. Темизация работает через CSS-переменные (`--paws-color-...`).

**Требования к реализации системы подсказок:**

1.  **Единый интерфейс (Props):**

    - Все интерактивные компоненты (`PawsButton`, `PawsCheckbox`, `PawsInput` и т.д.) должны принимать необязательный проп `tooltip` (тип: `string`).
    - _Пример:_ `<PawsCheckbox label="Legacy Mode" tooltip="Enables compatibility with older clients" />`.

2.  **Глобальное управление (Settings):**

    - Реализовать механизм глобального включения/выключения подсказок.
    - Библиотека должна экспортировать реактивное состояние или метод конфигурации (например, `setPawsUiConfig({ showTooltips: boolean })`), чтобы основное приложение Paws могло динамически управлять этим флагом на основе настроек пользователя.

3.  **Логика отображения (UX):**

    - **Задержка:** Подсказка должна появляться строго через 1 секунду (1000ms) после события `mouseenter`. Если курсор ушел раньше — таймер сбрасывается.
    - **Позиционирование:** Использовать стратегию, предотвращающую обрезание подсказки границами контейнера (рекомендуется Vue `Teleport` в `body` или использование библиотеки вроде `floating-ui`). Это критично для работы внутри iframe.
    - **Синглтон:** В любой момент времени на экране может быть только одна подсказка.

4.  **Стилизация и Темизация:**

    - Компонент подсказки должен быть стилизован с использованием _только_ CSS-переменных Paws (например, фон `--paws-color-bg-darkest`, текст `--paws-color-text-primary`, бордер/тень).
    - Никаких хардкодных цветов.

5.  **Архитектура:**
    - Рекомендуется использовать **Vue Directive** (например, `v-paws-tooltip`) или **Composable** внутри компонентов библиотеки, чтобы не дублировать логику таймеров в каждом компоненте.
    - Создать отдельный компонент `PawsTooltipContainer`, который будет отрисовывать само "облако" подсказки, принимая координаты и текст от директивы.
